Tous les codes doivent être dans le même dossier. Le fichier de données doit être dans le même dossier que tous les codes et au format csv utf-8.

La structure du fichier de données doit être la suivante : colonne 1 = date et heure, colonne 2 : production en Wh, colonne 3 : consommation en Wh, colonne 4 : prix en cts /kWh.

Le code à utiliser pour les simulations est "Model_a_utiliser", il faut le lancer et rentrer les informations demandées.
